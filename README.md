## ✒ 업로드 규칙
- 별도의 PR 없이 main에 직접 업로드 가능
- 파일의 경로는 'ch/item/item.md'
  ```
  ex) ch2/item2/item2.md
  ```
- 커밋 컨벤션
  - Docs : 문서, 이미지, 파일 추가
  - Fix : 문서, 이미지, 파일 변경
  ```java
  ex) Docs: item1 업로드 
  ```
